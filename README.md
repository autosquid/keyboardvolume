# keyboardvolume
control Mac Volumen with Alfred

An Alfred workflow to control system volume.

------------------

`v+` / `v++`/ `v+++` : VOLUME up by 10%/20%/30%

`v-` / `v--` /`v---`: VOLUME down by 10%/20%/30%

`v^` : maximum volume (100%)

`v_` : minimum volume (mute)

`v=` : mediate VOLUME (50% maximum)
